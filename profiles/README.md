# Conan profiles

Profile naming format:

fpr toolchain profiles:

  example: `gcc-8.2.0-linux-x86_64-toolchain`

  format: `<compier>-<compilerVersion>-<buildOS>-<target_architecture>-<toolchain>`

for other profiles:

  example: `gcc-8.2.0-linux-x86_64-toolchain`

  format: `<compier>-<compilerVersion>-<buildOS>-<target_architecture>-<toolchain>`
